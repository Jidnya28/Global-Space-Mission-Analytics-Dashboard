# Global Space Mission Analytics Project

## Project Overview

This project analyzes global space mission data to evaluate mission
performance, operational risk, and investment efficiency.

## Dataset Features

-   Year
-   Company
-   Rocket Type
-   Mission Status
-   Satellite Type
-   Rocket Cost
-   Mission Duration (Months)
-   Launch Site
-   Technology Used
-   Environmental Risk

## Objectives

1.  Measure mission success rate
2.  Compare company reliability
3.  Analyze cost efficiency
4.  Evaluate technology performance
5.  Support strategic decision-making

## Tools Used

-   Python (Pandas)
-   Power BI (for dashboard visualization)

## Key Insights

-   Reliability varies by organization
-   Innovation improves mission success
-   Smart spending outperforms high spending
-   Environmental conditions impact mission outcomes

## End Result

A decision-support dashboard that transforms historical mission data
into strategic intelligence.
